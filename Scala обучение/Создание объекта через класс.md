
#Scala 

Создание объекта не используя слово new, такой синтаксис называется "анонимный конструктор". Он работает для классов, у которых нет параметров в конструкторе.

Пример:

scalaCopy code

class Coffee:   
val price: Double = 2.0  
val cup = Coffee()`

Здесь `Coffee()` используется для создания экземпляра класса `Coffee`. Это возможно, так как `Coffee` не имеет параметров в своем конструкторе. Этот синтаксис более краткий и удобный, чем использование `new Coffee()`.

Однако, если у класса есть параметры в конструкторе, вы все равно должны использовать `new` для создания объекта:

scalaCopy code

class CoffeeWithSize(size: String):   
val price: Double = 2.0
val cup = new CoffeeWithSize("small")`

Как всегда, рекомендуется следовать советам и стилю программирования, принятым в вашей команде или проекте.
